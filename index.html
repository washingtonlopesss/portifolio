Sobre mim

Atualmente, sou Assistente de Sistemas na Nelson Wilians Advogados, mergulhando diariamente no emocionante universo do MySQL, contribuindo para a otimização e o desenvolvimento contínuo dos sistemas da NW.


Minhas habilidades atuais abrangem um sólido domínio das tecnologias front-end, como #HTML, CSS e #JavaScript, capacitando-me a criar interfaces de usuário intuitivas e visualmente atraentes. Além disso, tenho experiência no ecossistema .NET, demonstrando proficiência em C# e ASP.NET MVC com páginas Razor.


No mundo dos bancos de dados, sou proficientes em MySQL e SQL Server, o que me permite gerenciar e manipular dados com eficiência, garantindo a integridade e a segurança das informações.


Entretanto, minha jornada está longe de terminar! Estou em constante busca pelo aprimoramento profissional e, por isso, tenho me dedicado ao estudo do universo das APIs, incluindo as Minimal APIs ASP.NET e Web API ASP.NET. Meu objetivo é me tornar um desenvolvedor full stack, preparado para enfrentar os desafios do mercado de tecnologia com confiança e excelência.


Estou entusiasmado em conectar-me com profissionais da área para trocar ideias, colaborar em projetos desafiadores e expandir minha rede de contatos. Não hesite em entrar em contato comigo para oportunidades de colaboração, networking ou discussões técnicas!

Especialidades

  • Landing Pages

    Páginas modernas, pensadas para o usuário, construídas com foco em usabilidade e design intuitivo. As Landing Pages são projetadas para guiar o visitante em uma jornada clara e direcionada, seja para explorar um produto, conhecer um serviço ou realizar uma ação específica. Tenho a expertise necessária para desenvolver essas páginas de maneira eficiente, combinando estética e funcionalidade para oferecer uma experiência que realmente engaja o usuário.

  • APIs

    APIs são projetadas para permitir a integração eficiente entre sistemas, facilitando a troca de dados e a comunicação entre diferentes aplicações. Desenvolvo e mantenho APIs com o intuito de otimizar essa integração, garantindo que os sistemas funcionem de forma coordenada e eficaz. Tenho a expertise necessária para criar, gerenciar e integrar APIs, utilizando práticas de desenvolvimento sólidas para assegurar a funcionalidade e a confiabilidade das conexões entre sistemas.

  • Banco de Dados

    Um Banco de Dados bem estruturado garante a precisão e consistência dos dados, o que traz melhorias na eficiência operacional, na escalabilidade de um sistema e ainda facilita a tomada de decisões com base em dados. Tenho ampla experiência na gestão de Banco de Dados, desde de melhorias de performasse até a migração de dados entre sistemas, buscando sempre manter os dados íntegros e seguros.
